### Documentation: https://zer0-2.gitbook.io/type-chef-di

### Setup:

tsconfig.json:
```json
{
  "experimentalDecorators": true,
  "emitDecoratorMetadata": true,
  "target": "es6"
}
```
Install the npm package:

```sh
npm install type-chef-di
```
